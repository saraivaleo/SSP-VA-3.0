# SSP‑VA 3.0 – PWA (PT/EN + toggle de ultrassom)

## Novidades
- **Toggle de Ultrassom**: liga/desliga DSE, Língua espessa, HMDR (os checkboxes são desabilitados e as variáveis deixam de contribuir no cálculo quando OFF).
- **Toggle de Idioma (PT/EN)**: muda todos os rótulos e textos do app.
- **Branding simples** em `config.js` (brandName e textos).

## Como usar
1. Extraia o .zip e abra `index.html` (funciona offline).
2. Toque em **Ultrassom ON/OFF** no topo para incluir/ignorar variáveis de US.
3. Toque em **EN/PT** para alternar o idioma.

## Publicar (GitHub Pages)
- Suba os arquivos e ative Pages (main / root). Abra no celular e use “Instalar app”.
